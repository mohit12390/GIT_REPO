1. Setup a new repository

git remote add origin https://github.com/mohit12390/PythonRepo.git

git branch -M main

git push -u origin main

2. Adding the file to stage area

git add .

3. Commit the file to local repo

git commit -m "Readme file and ignore some useless file "

4. Push the change to remote repository

git push

5. check if any changes trackable for stage file .

git status

6. Check what all branch we are having

git branch

7. create an switch to new branch feature

git checkout -b feature

8. Switch to the main branch

git checkout main
